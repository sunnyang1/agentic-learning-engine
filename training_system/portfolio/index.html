项目背景

为影视飓风设计的
AI 驱动新人培训系统

MediaStorm 年产 130+ 影片，项目周期仅 5 天，编导淘汰率 ~10%。新人需要 14 天才能独立产出，导师时间被大量基础辅导占用。本系统通过 6 大 Agent 协作，实现个性化学习路径、自动作品预审、风险预警和季度评估。

🎯
核心目标
新人独立产出周期从 14 天 → 7 天
导师时间节省 30%+
课程完成率 >80%
编导淘汰率降至 5%
🛠
技术栈
飞书原生优先：多维表格 / 云文档 / 智能伙伴 / 应用引擎 / 机器人 / 审批流

LLM 供应商：阿里云百炼 (qwen) / 字节豆包 / Claude

基础设施：阿里云 OSS / 函数计算 / Redis
📊
关键指标
规则引擎延迟 < 200ms
LLM 推理延迟 < 3s
Agent 自主决策率 MVP 20% → 成熟期 85%
运营成本平稳期 ¥200-500/月
系统架构

四层 Agentic Learning 引擎

基于飞书原生能力构建，Orchestrator 统一调度，Memory 分层存储，6 大 Agent 协作执行，Tool Registry 标准化工具调用。

🧠
Orchestrator 调度中枢
飞书应用引擎 Workflow + 事件总线。接收学员行为事件（埋点/工作流/时间/状态变化/外部），通过规则层（80%）→ 策略层（15%）→ 模型层（5%）三层决策，调度单个/并行/串行 Agent 执行计划。
💾
Memory 记忆层
MVP 两层架构：Hot Memory（冻结快照 ~1,500 tokens，飞书多维表格 JSON 字段）+ Cold Storage（类型A多维表格/类型B云文档/类型C OSS 归档）。Phase 3 扩展 Procedural Memory + External Knowledge 语义 RAG。
🤖
6 大 Agent 智能体
诊断Agent：作品CT扫描 + 微场景测试
CoachAgent：苏格拉底式对话导师
内容Agent：动态学习路径组装（完成率>80%）
预测Agent：风险预警雷达（规则层→XGBoost模型）
实战Agent：项目工作流嵌入（MVP飞书消息推送）
AI Assessment：按题型分层准确率目标的出题器
🔧
Tool Registry
标准化工具注册表：Content Tools（课程/案例/资源搜索）、Assessment Tools（出题/评分/缺口分析）、Communication Tools（消息/会议/待办）、Project Tools（项目数据/工作流分析）、Analytics Tools（学习行为/团队能力/预测）。
核心模块

从入职到晋升的完整闭环

覆盖新人 Day 0-30 全周期，以及季度评估、等级认证、导师激励等人才发展体系。

🩺
诊断 & 测评
入职前48h收集作品 → Day 1 LLM多维度分析 → 3道微场景测试 → 生成能力缺口热力图 + 个性化新手村路径。AI Assessment Generator 按题型分层准确率目标出题。
🧑‍🏫
学习 & 辅导
Content Agent 每日早9点推送个性化套餐。Coach Agent 苏格拉底式对话，不直接给答案。实战 Agent 项目工作流嵌入。动态调整规则：连续高分→进阶内容，卡顿→简化版+1对1预警。
⚠️
预测 & 干预
硬阈值规则层：连续3天未登录/停留时间>200%/入职7天完成<60%。自动干预：关怀消息→导师通知→HR升级。Phase 2 升级为 XGBoost/LightGBM 轻量模型，目标命中率≥60%。
📋
季度 360° 评估
自评25% + 导师评价50% + Agent数据12.5% + 作品数据12.5%。飞书审批流实现：自评问卷 → 导师评价 → 自动汇总 → 季度成长报告。MVP去掉同事评价降低复杂度。
🏅
等级认证
L1助理→L2铜牌→L3银牌→L4金牌。能力雷达图+项目经历+带教记录+导师推荐+委员会审核。Agent自动评估达标状态，能力图谱"已发布"才触发晋升，防止误触发。
💎
导师激励
积分制：辅导1小时+10分 / 预审通过+2分 / 案例拆解+20分 / 学员晋升+50分。兑换：100分培训预算 / 500分调休 / 1000分年度大奖提名。月度/季度/年度荣誉体系。
